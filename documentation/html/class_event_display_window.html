Parametry
_eventReprezentowane wydarzenie.

sizeWskazuje, jak dużo informacji będzie wyświetlanych. Jeśli ten parametr ma wartość 1, to widoczny jest tylko tytuł wydarzenia, jeśli 2, wyświetlana jest także data rozpoczęcia, a jeśli 3, to dodatkowo widoczna jest data zakończenia.

◆ mousePressEvent()

void EventDisplayWindow::mousePressEvent ( QMouseEvent *  mouse_event)
protectedvirtual

Ta metoda nadpisuje metodę QWidget::mousePressEvent() z klasy bazowej.

Jest ona wywoływana, gdy wykryte zostanie kliknięcie myszką na dany obiekt. Jeśli nastąpiło kliknięcie lewym przyciskiem myszy, wywoływana jest metoda Event::runEditor() dla reprezentowanego wydarznia.
