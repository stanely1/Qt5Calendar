Opis szczegółowy

Klasa reprezentująca okienko wyświetlane przy próbie usunięcia wydarzenia.

Obiekty tej klasy są tworzone przez obiekt klasy EventEditorWindow, po wybraniu przez użytkownika opcji usuwania wydarzenia. Celem tej klasy jest potwierdzenie, że usunięcie danego wydarzenia jest celem użytkownika.
